Install theme

Once you've downloaded the installation file on ThemeForest, extract it and locate a file called exploore.zip. You can install the theme by using one of two installation methods:

1. Wordpress upload

Step 1: Go to Admin Panel → Appearance → Themes → Add New → Upload Theme.

Step 2: Browse to file exploore.zip and click "Install Now".

2. FTP upload

- Upload non-zipped theme folder called exploore to "/wp-content/themes/" folder in your WordPress installation folder on your server.

3. Active Theme: Go to Admin Panel → Appearance → Themes, locate the theme and click "Activate" button to activate the theme.

You will see a notification message to change permalink setting to post name. Please do this.

Import Demo Content

You can either start building your site from scratch, or import one of our demo sites and then modify it to fit your needs. We will now explain how to perform the import process.
First, you must clear all available data which you created before (posts, pages, menu and widgets) and make sure you activated Slz Core plugin then do the following one of two ways:
1. One Click Importer: Please go to Admin Panel → Theme options → Demo Inporter. Click "Import Demo" button and wait for a while. When progress go too long ( > 15 mins ), please try to refresh page and re-import.
2. XML Importer: Use other tools in download package, please read "Demo-content / readme.txt"

After you import demo content, please go to Admin Panel → Settings → Permalink and click Save Changes button to complete process.
Once the demo data has been imported, you can make changes according to your requirements and continue building your site.

Accommodations



Accommodations - Custom Post Type

Accommodations is a custom post type and so you can manage it like post. Accommodation has several options in Accommodation Options metabox like bellow.

Go to Accommodation → Add New in Admin Panel to see how Accommodation Options metabox displays.

General Tab
  • Display Title: You can enter accommodation custom title which display on front-end. If this metabox empty, this value will receive value of title.
  • Status: Choose a status for accommodation. You can disable accommodation that accommodation will not display on front end by setting in Theme options → Accommodations Setting → Accommodations Status Disable.
  • Description: This metabox allows you to enter description for accommodation.
  • Thumbnail: You can upload image which display as feature image of accommodation on front-end.
  • Check in time & Check out time: You can enter information about time check in / check out accommodation. This field also will be used on search and sorting.
  • Tax Rate: This field will be used in price calculation.
  • Star Rating: You can enter number of star of accommodation in this metabox. This field also will be used on search and sorting.
  • Average Price: This field will be used as a presentation price and also will be used on search and sorting.
  • Email & Phone & Address: This metaboxes will be used to display contact information of accommodation.
Room Types Tab

You can show or hide Room Types Box in detail page by Show Room Types Box option. Room Types Box will be hidden if Room Types of accommodation are empty. You can also create new room type by Add new room type link.

Discount Tab
  • Is Discount: This metabox allows you to show discount of accommodation.
  • Discount Rate: You can enter number percent which will be discount. Discount in front-end will be hidden if discount rate are empty even though is discount is checked.
  • Discount Text: You can enter a text which will be shown with discount rate.
  • Discount Start Date & Discount End Date: This fields will be used to limit discount period.
Others Tab
  • Booking Email: This email will receive user booking information when this accommodation booked. Emails are separated by commas.
  • Mark the accommodation as featured: Mark your accommodation as Is Featured if you want to be able to show this accommodation in accommodation lists (widgets and custom pages) where "Show only featured" flag is checked.
  • Attachments: You can upload attach documents that related to the accommodation.
Deposit Tab

This tab allows you to display deposit information of this accommodation on front-end when user booking. Go to Deposit to see how it works.

  • Short Description: This metabox allows you to enter description for tour.
  • Date Type: Choose date type for the tour. Check Fixed Days if the tour open with fixed days as weekly, monthly, any day during the season or a specific date. Example:
    • If you choose Date Frequency is Weekly and Day Of Week is Mon, this tour is only available for booking every monday.
    • If you choose Date Frequency is Monthly and Day Of Month is 20, this tour is only available for booking 20th of every month.
    • If you choose Date Frequency is Season and Tour Date is from 2016-09-20 to 2016-09-30, this tour is available for booking any day from 2016-09-20 to 2016-09-30.
    • If you choose Date Frequency is Specific Day and Tour Date is 2016-09-20 to 2016-09-30, this tour is only available for booking on 2016-09-20.
  • Duration: You can enter information about duration of tour.
  • Available Seats: You can enter available seats of tour.

    • If you choose Date Frequency is Season and Tour Date is from 2016-09-20 to 2016-09-30, this cruise is available for booking any day from 2016-09-20 to 2016-09-30.
    • If you choose Date Frequency is Specific Day and Tour Date is 2016-09-20 to 2016-09-30, this cruise is only available for booking on 2016-09-20.
  • Duration: You can enter information about duration of cruise.
  • Price Per Adult / Price Per Child: This field will be used as a presentation price and also will be used on search and sorting.
  • Available Seats: You can enter available seats of cruise.
  • Destination: You can enter destination of cruise.
  • Is Discount: This metabox allows you to show discount of cruise.
  • Discount Rate: You can enter number percent which will be discount. Discount in front-end will be hidden if discount rate are empty even though is discount is checked.
  • Discount Text: You can enter a text which will be shown with discount rate.
  • Status: You can choose a status for cruise.

Translation


Translate the Theme to your language

If you want to translate the theme to other language (e.g. for the Portuguese spoken in Brazil ) you must:

  • Open wp-config.php and edit: define(‘WPLANG’, "pt_BR");
  • Go into wp-content/themes/exploore/languages folder and replace exploore.mo and exploore.po to pt_BR.mo and pt_BR.po
  • Download Poedit and open .po file by Poedit

  • Choose text which want to translte and enter translated text into Translation box.
  • After you have done your translation, click Save.
  • Once all settings have been completed, you will see translated text displayed as below.

In Admin Panel, go on Settings → General, choose Site Language is your translated language.

You can learn more about translating WordPress here Installing WordPress in Your Language

Note : After you translate your pages you need to go into Settings → Permalinks and click on Save.



Translate the Plugin to your language

If you want to translate the plugin to other language (e.g. for the Portuguese spoken in Brazil ) you must:

  • Make sure wp-config.php defined this code: define(‘WPLANG’, "pt_BR");
  • Go into wp-content/plugins/slz-core/languages folder and replace slz-core.pot to slz-core-pt_BR.po
  • Open .po file by Poedit.

  • Choose text which want to translte and enter translated text into Translation box.
  • After you have done your translation, click Save. Once your slz-core-pt_BR.po file saved, slz-core-pt_BR.mo file will created automatically.


Create PO file with POEDIT

In Poedit window, choose File → New to create a new po file. Choose "Extract from source" to configure source code extraction.

In "Translation properties" of Catalog properties dialog, please choose Source code charset is UTF-8.

In "Sources paths" tab, please add folder which you want create po file.
Example :
- Choose "wp-content/themes/exploore" if you want to translate Exploore theme.
- Choose "wp-content/plugins/slz-core" if you want to translate Slz-core plugin.

Next switch to "Sources keywords" tab, press the same "New item" button and add all Gettext functions you use for your plugin as below.

Make sure all necessary functions are added and press OK button. After your po file is nicely saved, navigate to Catalog and choose to "Update from sources"

- Choose text which want translate and translate them to your language.

- After you have done your translation, click Save.
